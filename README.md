# ClaudeUsage Review Archive

This repository is archived.

Work has moved to the clean-room, MIT-licensed Pitwall repository:

https://github.com/GeorgeQLe/pitwall

## Why This Repository Is Archived

This project began as a fork of `linuxlewis/claude-usage` and contains upstream-derived Swift menu bar app code from that fork lineage. The upstream repository did not include a clear top-level open-source license at the time of review, so this repository should be treated as historical prototype/reference material only.

Do not use this repository as the commercial or open-source distribution path for Pitwall.

## Current Direction

Pitwall is being rebuilt as a clean-room implementation in the new repository. The new repo includes:

- MIT license from day one
- clean-room implementation rules
- macOS reproduction spec
- reproduction checklist
- static landing page

Use `GeorgeQLe/pitwall` for all future product, implementation, website, and release work.
